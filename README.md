# Sales Team Management Dashboard

A full-stack **Sales Team Management Dashboard** built with **React + Vite + TypeScript** on the frontend and **Express.js** on the backend.  
This project is designed to help sales teams and managers track performance, monitor targets, manage calls, leads, reports, and view analytics in one centralized dashboard.

---

## 🚀 Features

- **Secure Login System**
  - Role-based login for Admin and team members
  - Session-based authentication

- **Dashboard Overview**
  - Team performance metrics
  - Sales target progress
  - Reports & analytics view
  - Quick insights for managers and executives

- **Sales Team Management**
  - Manage sales executives and team performance
  - View assigned targets and progress
  - Track calls, leads, and conversions

- **Reports & Analytics**
  - Sales performance reporting
  - Data-driven insights for decision making
  - Team-wise performance summaries

- **Backend API**
  - Authentication APIs
  - Session management
  - Sales/team data endpoints
  - JSON-based local data storage
